Creating hubs and naming them

Using the root global hub is just a sugar. You can create your own hub:

var myHub = new HUBU.Hub();

Moreover you can name this hub:

var myHub = new HUBU.Hub()
    .configure(null, {component_name: "my hub is bigger than yours"}).start();

If you wonder about the first argument of the configure method, it's used to pass a parent hub to create hirearchical composition. This feature is coming soon.

All created hubs have their own services and events. So they cannot be shared between hubs. A component can be registered only on one hub.
