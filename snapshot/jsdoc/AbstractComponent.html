AbstractComponent

new AbstractComponent()

Abstract Component class. This used is not intended to be used directly, and is just here for documentation purpose. Indeed, the returned object contains the four required methods that **all** component must have. Any Javascript object with those 4 methods can be considered as a valid component. The 4 required methods are: - `getComponentName()` : return the default component name - `configurate(hub, [configuration])` : configures the component - `start()` / `stop()` : called when the component is started / stopped Returned objects do not intend to be used, they are just mock / empty instances.
